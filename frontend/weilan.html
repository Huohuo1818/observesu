微澜系列

属于年轻人的情绪微醺。以 12% Vol. 的轻盈，承接日常的疲惫与紧绷。 不借酒消愁，只在花果与草木的呼吸间，把情绪慢慢安放。

01 / 06

酸枣仁玫瑰酒

暮色

给夜晚的一点温柔，让情绪慢慢落回身体里。

暮色写给不必热闹的夜晚。玫瑰先打开情绪，桑椹承接入口的浆果酸甜， 酸枣仁把酒体往后收，留下安静、柔软、很东方的尾韵。

风味层次宝石红酒液，花香柔和。微酸轻甜，关键不在于甜，而在于入口的“软”。

草本初衷酸枣仁定下安静的气质，重瓣红玫瑰负责馥郁花香，桑椹的浆果气息让草本变得更易亲近。

情绪场景精神消耗过后的独处时刻。帮助紧绷的神经从兴奋切换至放松，给夜晚一点温柔的过渡。

核心草木酸枣仁 / 重瓣红玫瑰 / 桑椹

草木寻迹山野枣林 / 玫瑰花田

酒体视觉宝石红透，轻盈如珀

岁月沉淀鲜果浸润，花酿慢萃

酒精度12% Vol.

01 / 07

黑枸杞人参酒

晨露

清冽、明亮，像清晨之前先醒来的那口呼吸。

晨露有更挺拔的清醒感。黑枸杞带来冷调紫色与清透浆果气息， 人参不做厚重参味，只在尾调留下一丝草木回甘，桂圆把锐气收圆。

风味层次迷人的紫罗兰调酒液，入口清冽干净。前段轻盈，后段随着草木的回甘慢慢舒展。

草本初衷黑枸杞定格冷调浆果色泽，长白山人参不留厚重参味、只留草木清香，桂圆肉将整体锐气温润收圆。

情绪场景熬夜后的疲倦早晨，或长期睡眠不足时的状态回充。清醒，明亮，拂去疲惫感。

核心草木黑枸杞 / 林下参 / 桂圆肉

草木寻迹柴达木盆地 / 长白山脉

酒体视觉紫罗兰调，清澈流光

岁月沉淀鲜参原浸，冷萃取色

酒精度12% Vol.

适用器皿细长直身杯 / 香槟杯

01 / 07

紫苏青梅陈皮酒

入林

像一口喝下潮湿的风、青色的果与林间草木的呼吸。

入林是微澜里最有空间感的一支。紫苏带来冷冽草本香，青梅把酸感拉直， 陈皮把尾调压得更圆润，形成清爽、明快、适合反复入口的节奏。

风味层次清透酸香，草本微苦与明亮回甘互相咬合。层次丰富，宛如一杯出自现代酒吧的东方特调。

草本初衷新鲜紫苏建立极具辨识度的冷冽香气，潮汕青梅带来明亮酸度，新会陈皮则完美衔接了前段的清新与后段的温润。

情绪场景缓解外卖或重油饮食后的肠胃负担，疏解郁闷。适合久坐后的自我犒劳，或夏日聚餐的解腻分享。

核心草木新鲜紫苏 / 潮汕青梅 / 新会陈皮

草木寻迹岭南梅林 / 新会橘园

酒体视觉青梅浅黄，清透微光

岁月沉淀果木同酿，陈化交融

酒精度12% Vol.

适用器皿极简敞口杯 / 威士忌古典杯
